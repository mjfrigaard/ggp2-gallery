Should I use this graph?


Line graphs display changes of a numeric quantitative variable (on the y axis) over time (on the x axis). Color, line style, and faceting can be used to show group changes for categorical variables.

PACKAGES:

Install packages.

Code 
install.packages("fivethirtyeight")

library(ggplot2)

DATA:

We’ll be using the US_births_1994_2003 and US_births_2000_2014 datasets from the fivethirtyeight package:

  • Binding these together (they have identical columns)

  • Create a day_category variable that distinguishes between weekdays (Weekends) and weekends (Weekday)

  • Use year and month to create yr_mnth

  • Use year and quarter to create yr_qtr

Save these changes to usbirth_1994_2014:

Code 
usbirth_1994_2014 <- fivethirtyeight::US_births_2000_2014 |>
    dplyr::bind_rows(fivethirtyeight::US_births_1994_2003) |> 
    dplyr::mutate(
        day_category = case_when(
            day_of_week %in% c("Sun", "Sat") ~ "Weekend",
            day_of_week %nin% c("Sun", "Sat") ~ "Weekday",
            TRUE ~ NA_character_
        ),
        month = dplyr::if_else(
            condition = month < 10,
            true = paste0("0", month),
            false = as.character(month)
        ),
        yr_mnth = paste0(year, "-", month),
        yr_mnth = lubridate::ym(yr_mnth),
        yr_qtr = paste0(lubridate::year(date),
                              "/0", 
                              quarter(date)),
        yr_qtr = factor(yr_qtr, ordered = TRUE)
    )

LABELS:

In the previous graph, we can see the number of average births reaches it’s peak in quarter three of 2008.

We’re going to create a small table of labels, label_dow, that we’ll use to label the maximum and minimum values. label_dow will have 7 rows (one for each day of the week) and a label variable (lbl) which we can use with geom_label()

Code 
label_dow <- usbirth_1994_2014 |>
    dplyr::group_by(day_of_week) |>
    dplyr::summarise(
        max_births = max(births, na.rm = TRUE)) |>
    dplyr::ungroup() |>
    dplyr::arrange(desc(max_births)) |>
    dplyr::rename(day = day_of_week) |>
    dplyr::mutate(lbl = paste0(day, 
                               ", max births = ", 
                               max_births)) |>
    dplyr::inner_join(y = usbirth_1994_2014, 
                      by = c("max_births" = "births")) |>
    dplyr::select(day, max_births, lbl, 
        date, day_category, yr_mnth, yr_qtr)
label_dow

Use filter() to get maximum and minimum values for "Wed" and "Tues" (in label_max_wed_dow and label_min_tues_dow):

We'll be using the `US_births_1994_2003` and `US_births_2000_2014` datasets from the `fivethirtyeight` package: 

  - Binding these together (they have identical columns)  

  - Create a `day_category` variable that distinguishes between weekdays (`Weekends`) and weekends (`Weekday`)  

  - Use `year` and `month`  to create `yr_mnth`  

  - Use `year` and `quarter` to create `yr_qtr` 

Save these changes to `usbirth_1994_2014`:

::: {style="font-size: 0.75em;"}
```{r}
#| label: data_usbirth_1994_2014
#| eval: true
#| echo: true
usbirth_1994_2014 <- fivethirtyeight::US_births_2000_2014 |>
    dplyr::bind_rows(fivethirtyeight::US_births_1994_2003) |> 
    dplyr::mutate(
        day_category = case_when(
            day_of_week %in% c("Sun", "Sat") ~ "Weekend",
            day_of_week %nin% c("Sun", "Sat") ~ "Weekday",
            TRUE ~ NA_character_
        ),
        month = dplyr::if_else(
            condition = month < 10,
            true = paste0("0", month),
            false = as.character(month)
        ),
        yr_mnth = paste0(year, "-", month),
        yr_mnth = lubridate::ym(yr_mnth),
        yr_qtr = paste0(lubridate::year(date),
                              "/0", 
                              quarter(date)),
        yr_qtr = factor(yr_qtr, ordered = TRUE)
    )
dplyr::glimpse(usbirth_1994_2014)
```

We're going to create a small table of labels, `label_dow`, that we'll use to label the maximum and minimum values. `label_dow` will have 7 rows (one for each day of the week) and a label variable (`lbl`) which we can use with `geom_label()`

```{r}
#| label: label_dow
#| code-fold: true
#| eval: true
#| echo: true
#| warning: false
#| message: false
#| out-height: '100%'
#| out-width: '100%'
#| column: page-inset-right
#| layout-nrow: 1
label_dow <- usbirth_1994_2014 |>
    dplyr::group_by(day_of_week) |>
    dplyr::summarise(
        max_births = max(births, na.rm = TRUE)) |>
    dplyr::ungroup() |>
    dplyr::arrange(desc(max_births)) |>
    dplyr::rename(day = day_of_week) |>
    dplyr::mutate(lbl = paste0(day, 
                               ", max births = ", 
                               max_births)) |>
    dplyr::inner_join(y = usbirth_1994_2014, 
                      by = c("max_births" = "births")) |>
    dplyr::select(day, max_births, lbl, 
        date, day_category, yr_mnth, yr_qtr)
```
