ARGUMENTS:

Below the graphs are built with geom_path()’s lineend argument set to "butt" and the three possible options for linejoin ("bevel", "round", and "mitre"). The size and linemitre have been increased to exaggerate their effects.

Code 
ggp2_path_bb <- ggplot(data = economics,
    mapping = aes(x = date, y = unemploy)) +
    geom_path(
        size = 5,
        lineend = "butt",
        linejoin = "bevel",
        linemitre = 3) + 
    labs(title = "butt/bevel")
