Mosaic Plots

Graph info

When would I use this graph?


A mosaic plot is similar to a stacked bar-graph, but instead of only relying on height and color to display the relative amount for each value, mosaic plots also use width.

We can build mosaic plots using the ggmosaic package.

  • Packages
  • Data

PACKAGES:

Install packages.

install.packages("fivethirtyeight")
library(fivethirtyeight) 
devtools::install_github("haleyjeppson/ggmosaic")
library(ggmosaic)
library(ggplot2)

DATA:

For this graph, we’ll be using the fivethirtyeight::flying dataset, after removing the missing values from baby and unruly_child.

flying <- fivethirtyeight::flying
fly_mosaic <- filter(flying, 
              !is.na(baby) & !is.na(unruly_child)) |> 
              select(baby, unruly_child)
glimpse(fly_mosaic)

CODE:

Create the labels

Map the product() of unruly_child and baby to the x axis

Map baby to fill

Add theme_mosaic()

labs_mosaic <- labs(
      title = "In general...", 
      subtitle = "...is it rude to...",
      x = "... bring a baby on a plane?",
      y = "..,knowingly bring unruly children on a plane?",
      fill = "Responses") 
ggp2_mosaic <- ggplot(data = fly_mosaic) +
  geom_mosaic(aes(x = product(unruly_child, baby), 
      fill = baby)) +
  theme_mosaic()
ggp2_mosaic + 
    labs_mosaic

NOTES:

It’s a good idea to read the ggmosaic vignette, particularly the section on ordering and conditioning.

For example, if we set the product() variable as unruly_child and the conditional variable as baby, we see a very different display than if we follow the stanard two-variable syntax:

# build conditional mosiac plot
labs_mosaic_cond <- labs(
  subtitle = "Conditional plot",
  x = "..,knowingly bring unruly children on a plane?",
  y = "... bring a baby on a plane?",
  fill = "Responses")
ggp2_mosaic_cond <- ggplot(data = fly_mosaic) +
  geom_mosaic(aes(
    x = product(unruly_child), # product variable
    fill = unruly_child,
    conds = product(baby))) + # conditional variable
  theme_mosaic(base_size = 9)
mosaic_cond <- ggp2_mosaic_cond + labs_mosaic_cond
# build 2-variable mosiac plot
labs_mosaic_2var <- labs(
  subtitle = "2-Variable plot",
  x = "..,knowingly bring unruly children on a plane?",
  y = "... bring a baby on a plane?",
  fill = "Responses")
ggp2_mosaic_2var <- ggplot(data = fly_mosaic) +
  geom_mosaic(aes(
    x = product(unruly_child, baby),
    fill = unruly_child)) +
  theme_mosaic(base_size = 9)
mosaic_2var <- ggp2_mosaic_2var + labs_mosaic_2var
# patch both together
patchwork::plot_spacer() + # pad the area in front of the plot 1
  mosaic_cond +
  patchwork::plot_spacer() + # pad the area in front of the plot 2
  mosaic_2var +
  patchwork::plot_layout(
    # limit to 2 rows
    nrow = 2,
    # make plots in column 2 5x the width of column 1
    widths = c(1, 5)) +
  patchwork::plot_annotation(
    # adjust title text size
    theme = theme_mosaic(base_size = 11),
    title = "In general...is it rude to...")
